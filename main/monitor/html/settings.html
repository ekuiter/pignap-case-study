PIGNAP-WLAN

Das PIGNAP-WLAN kann mit einem Passwort abgesichert werden.

Lassen Sie die folgenden Felder leer, um das aktuelle Passwort beizubehalten. Falls das Passwort geändert wird, müssen Sie sich ggf. neu im WLAN anmelden.
